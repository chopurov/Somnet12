Раз вы уже создали HTML-квиз, выложить его в интернет через GitHub очень просто — это бесплатно и делается за несколько минут.

Вот пошаговая инструкция, как разместить ваш файл quiz.html с помощью сервиса GitHub Pages.

Шаг 1: Создайте репозиторий на GitHub

  1. Войдите в свой аккаунт на github.com.

  2. Нажмите на зеленую кнопку "New" (или + в правом верхнем углу → "New repository").

  3. В поле "Repository name" есть два варианта:

    • Личный сайт: Назовите его ваш-username.github.io (например, john-doe.github.io). После этого ваш сайт откроется по адресу https://ваш-username.github.io.

    • Проект/портфолио: Назовите как угодно, например, my-quiz. Сайт будет доступен по адресу https://ваш-username.github.io/my-quiz/.

  4. Обязательно поставьте переключатель "Public" (для бесплатного хостинга).

  5. Нажмите "Create repository".-3-8

Шаг 2: Загрузите ваш HTML-файл

Теперь нужно добавить ваш файл quiz.html в созданную папку (репозиторий).

  1. В открывшемся репозитории нажмите "Add file" → "Upload files".

  2. Перетащите мышкой ваш HTML-файл с компьютера в появившуюся область.

  3. Важно: Если у вас файл называется не index.html, а, например, quiz.html, это нормально. Просто учтите это при открытии сайта.

  4. Внизу нажмите зеленую кнопку "Commit changes".-8

Шаг 3: Активируйте GitHub Pages (самое главное!)

Сейчас файлы просто лежат на сервере, но сайт еще не работает. Нужно включить публикацию.

  1. В вашем репозитории перейдите во вкладку Settings (настройки).

  2. В левом меню найдите раздел Pages (это в группе "Code and automation").

  3. В разделе "Branch" выберите ветку main (или master, смотря что у вас) и оставьте папку / (root).

  4. Нажмите кнопку "Save".-1-7

Шаг 4: Откройте ваш сайт

  1. Подождите 1-2 минуты, пока GitHub подготовит страницы.

  2. Вернитесь в тот же раздел Settings → Pages. Там появится зеленая строка с вашим адресом.

  3. Перейдите по ссылке.

    • Если репозиторий назывался username.github.io, то файл quiz.html откроется автоматически, и вы увидите ваш квиз.

    • Если репозиторий назывался иначе (например, my-quiz), то в адресной строке пропишите: https://ваш-username.github.io/my-quiz/quiz.html.

Как сделать сайт ещё лучше?

  1. Переименуйте файл в index.html. Это стандартное имя главной страницы. Если переименовать ваш файл в index.html, то сайт будет открываться автоматически, без дописывания /quiz.html в конце. Это самый удобный вариант для пользователей.-3

  2. Подключите свой домен. Если у вас есть купленное имя (например, my-site.ru), его можно привязать.

    • В настройках Settings → Pages в поле "Custom domain" введите ваш адрес.

    • В настройках вашего домена (там, где вы его покупали) нужно создать CNAME-запись, которая указывает на ваш-username.github.io.-1-2
